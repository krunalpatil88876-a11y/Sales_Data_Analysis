# 📊 Sales Data Analysis System

## 💡 Overview

This project focuses on analyzing a large-scale retail dataset (~50,000+ records) to extract meaningful business insights. It demonstrates core data science skills including data cleaning, exploratory data analysis (EDA), visualization, and insight generation.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze sales and profit trends
* Identify high-performing regions and products
* Detect loss-making transactions
* Generate actionable business insights

---

## 📂 Dataset Information

* **Domain:** Retail / Sales
* **Rows:** ~51,000+
* **Columns:** 25+
* **Key Features:**

  * Sales, Profit, Quantity
  * Region, Category, Sub-Category
  * Order Date, Ship Date
  * Customer and Product details

---

## ⚙️ Project Workflow

1. **Data Loading**

   * Imported dataset using Pandas

2. **Data Cleaning**

   * Removed missing values
   * Eliminated duplicate records
   * Converted date columns

3. **Exploratory Data Analysis (EDA)**

   * Statistical summary
   * Correlation analysis
   * Distribution analysis

4. **Data Visualization**

   * Sales by region
   * Profit by product
   * Sales trends over time
   * Correlation heatmap

5. **Business Insights**

   * Identified top-performing regions
   * Analyzed profit trends
   * Detected loss-making transactions
   * Calculated overall sales performance

6. **Export Results**

   * Saved cleaned dataset for further use

---

## 📊 Key Insights

* 💰 Total Sales: High overall revenue (~12M+)
* 📈 Profit Margin: Moderate (~11–12%)
* 🏆 Best Region: Central region shows highest performance
* ⚠️ Losses: Some transactions have negative profit
* 📉 Variability: Sales distribution shows high variance
